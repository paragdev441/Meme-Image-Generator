## About

Meme Generator lets you:
1. Fetch random memes image.
2. Add text to the Meme image.
3. Add styles to the text like font family, font style, width, height, color, etc.

## Setup for running the development version

1. Clone this repository.

2. Open the terminal on the `Meme-Image-Generator` folder & install node modules `npm i`.
3. Run `npm start`.
